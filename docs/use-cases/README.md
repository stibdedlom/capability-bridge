# Use cases

This directory records agent-centered use cases for the Capability Bridge.

## Near-term (V0–V1)

- Project resume + task routing
- Voice-initiated intent framing
- Visible Worker + Reviewer pair
- Mobile approval and interruption
- Bounded agent loop with stop conditions
- Session health monitor

## Mid-term (V2–V4)

- Persistent agent rooms per project
- Review board mode
- Cross-project portfolio orchestration
- Memory promotion pipeline
- Trace replay / eval lab
- Local/cloud model router

## Speculative (V5+)

- Autonomous maintenance mode
- Emergency stop and rollback coalition
- Client-mode cross-org orchestration
- Spatial command center on Vision Pro
- Predictive attention routing

See the detailed taxonomy in `projects/stibdedlom-ideation-workshop/voice-visible-session-orchestration-research.md`.
